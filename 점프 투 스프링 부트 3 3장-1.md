
## 1. 페이징 메커니즘: Pageable과 Page

Spring Data JPA에서 페이징은 '질문'과 '답변'의 구조로 이루어집니다.

* **Pageable (질문):** 클라이언트가 서버에 요청하는 데이터의 **규격**입니다. "몇 번째 페이지인가?", "한 번에 몇 개씩 가져올 것인가?", "어떤 기준으로 정렬할 것인가?"에 대한 정보를 추상화한 인터페이스입니다.
* **Page (답변):** 서버가 클라이언트에게 돌려주는 **결과물**입니다. 실제 데이터 목록뿐만 아니라 전체 페이지 수, 전체 데이터 개수, 현재 페이지 번호와 같은 **메타데이터**를 포함합니다.

> **핵심 개념:** `Page` 객체를 반환하면 프론트엔드는 별도의 추가 계산 없이도 전체 페이지 수 등을 파악해 UI(페이지네이션 버튼 등)를 바로 렌더링할 수 있습니다.

## 2. 데이터 흐름의 경계: Entity와 DTO

데이터는 계층을 이동할 때 그 목적에 맞는 옷(객체)으로 갈아입어야 합니다. 이를 **계층 간 데이터 전송 객체(DTO) 분리**라고 합니다.

* **Entity (내부용):** 데이터베이스 테이블과 직접 연결된 설계도입니다. 시스템의 가장 핵심적인 비즈니스 규칙을 담고 있으며, 외부로 노출될 경우 DB 구조가 드러나는 보안 리스크가 있습니다.
* **DTO (외부용):** API 응답 전용 객체입니다. 클라이언트가 필요로 하는 데이터만 선별하여 담습니다.

### 왜 서비스(Service)에서 변환하는가?

1. **영속성 계층의 격리:** 컨트롤러가 DB 상세 구조를 알 필요가 없게 만듭니다.
2. **지연 로딩(Lazy Loading) 처리:** JPA의 지연 로딩은 트랜잭션 내(Service)에서만 안전하게 작동하므로, 컨트롤러로 넘기기 전에 필요한 데이터를 미리 DTO에 채워넣어야 에러가 발생하지 않습니다.
->정리하자면
지연 로딩은 "나중에 데이터 필요할 때 DB에 또 물어볼게"라는 약속입니다.
하지만 그 약속은 서비스 계층(트랜잭션 안)에서만 유효합니다.
컨트롤러로 나가면 DB에 물어볼 방법이 사라집니다.
따라서, 연결이 살아있을 때 필요한 걸 미리 다 꺼내서 순수한 데이터 덩어리(DTO)에 담아 보내야 안전합니다.

## 3. 보안 필터링 원리: Spring Security

Spring Security는 애플리케이션의 입구에서 요청을 검사하는 **필터(Filter)** 역할을 수행합니다.

* **화이트리스트(PermitAll):** 로그인이 없어도 누구나 접근 가능한 경로입니다. (예: 목록 조회, 회원가입)
* **인증 필수(Authenticated):** 유효한 세션이나 토큰이 있는 사용자만 접근 가능한 경로입니다. (예: 글쓰기, 삭제)
* **CSRF 보호:** REST API는 세션이 아닌 무상태(Stateless) 방식(JWT 등)을 주로 쓰기 때문에, 외부 호출을 허용하기 위해 CSRF 설정을 끄는 것이 일반적입니다.

## 4. 계층별 핵심 요약 (구조도)

| 계층 (Layer) | 주요 역할 | 취급 데이터 |
| --- | --- | --- |
| **Controller** | API 입구, HTTP 요청 처리 및 응답 반환 | **DTO** |
| **Service** | **비즈니스 로직**, Entity ↔ DTO 변환 | **Entity & DTO** |
| **Repository** | DB 쿼리 실행, 영속성 관리 | **Entity** |

### 요약된 흐름 예시

1. **요청:** `Controller`가 `page` 번호를 받아 `Service` 호출.
2. **조회:** `Service`가 `Pageable`을 생성해 `Repository`에서 `Page<Entity>` 조회.
3. **변환:** `Service`가 `Page<Entity>`를 `.map()`으로 `Page<DTO>`로 변환.
4. **응답:** `Controller`가 최종적으로 `Page<DTO>`를 JSON으로 반환.

---

이 구조는 시스템의 유지보수성을 높이고, DB의 변경이 클라이언트에게 직접적인 영향을 주지 않도록 차단하는 **캡슐화**의 전형적인 모델입니다.
